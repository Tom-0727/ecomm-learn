# 中国电商实战课

**系统掌握从需求洞察到复购留存的完整电商业务链路**

## 课程特色

- 📚 **10大核心模块** — 覆盖需求洞察、选品策略、供应链、平台搭建、内容运营、流量获取、转化优化、履约物流、口碑管理、复购留存
- 📝 **160道专业题库** — 三级难度，含单选、多选、判断、场景计算题
- 🎯 **自适应考试系统** — 根据历史答题弱项动态调整出题比例
- 🔬 **项目诊断训练** — 8个真实业务场景，练习「判断项目卡在哪一环」
- 📊 **指标速查手册** — 全链路核心指标健康基准、计算公式一表呈现

## 访问方式

### 本地访问
```bash
bash start-server.sh
# 打开 http://localhost:8080
```

### GitHub Pages（在线访问）
部署后直接访问：`https://<your-username>.github.io/<repo-name>/`

## 网站结构

```
website/
  index.html        # 首页：课程地图 + 进度概览
  course.html       # 课程：10模块带侧边目录
  exam.html         # 考试：160题自适应系统
  diagnose.html     # 诊断：8个项目卡点案例
  metrics.html      # 指标：全链路速查手册
  js/
    courseData.js   # 课程内容数据
    examData.js     # 题库（160题）
    diagnoseData.js # 诊断案例数据
    app.js          # 进度追踪 + localStorage
    exam.js         # 自适应考试引擎
  css/
    styles.css      # 全站样式
```

## 学习路径建议

1. **首页** → 了解整体链路和自己的学习进度
2. **课程页** → 按顺序学习10个模块（约7小时总时长）
3. **考试** → 学完3-4个模块后开始测试，系统会标注弱项
4. **弱项强化** → 针对低分模块重点复习
5. **项目诊断** → 学完全部模块后挑战8个诊断案例
6. **指标速查** → 工作中随时查阅参考基准
